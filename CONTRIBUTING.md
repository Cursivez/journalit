# Contributing to Journalit

Journalit is public-source for transparency and Obsidian Community plugin review. The public repository is not the primary development repository and is not maintained as a general community contribution project.

The private Journalit plugin repository remains the source of truth. Changes accepted from public discussion or patches may be reimplemented, cherry-picked, or adapted in the private repository before being mirrored here.

## Preferred support channel

For bugs, questions, feature requests, and trading-workflow feedback, please use Discord first:

https://discord.gg/AkSw3D9h8b

Traders should not need to prepare perfect GitHub reports or patches. If something is broken, confusing, or missing, tell us what happened and we will handle the product and implementation work.

## Issues and pull requests

To keep the public repository focused and low-noise:

- Issues from unapproved contributors may be closed automatically.
- Pull requests from unapproved contributors may be closed automatically.
- Maintainers may reopen, approve, or copy useful reports into the private development process.
- Public pull requests are treated as proposed patches, not as the source of truth.

Maintainers can approve future participation by commenting:

- `lgtmi` — approve future issues.
- `lgtm` — approve future issues and pull requests.

Please do not submit large pull requests without prior discussion.
