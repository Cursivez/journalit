# **[1.7.2] - 2026/04/22**
Navigation, drawdown analytics, and trade journaling improvements, with major quality-of-life fixes across dashboard and filters.

### **Navigation Sidebar + Journalit search Bar**
![Sidebar](https://github.com/user-attachments/assets/f5c2727d-f0e0-4643-a125-b89a63c2be7a)
![Search Bar](https://github.com/user-attachments/assets/c3187b34-ff76-45e7-a285-243d7781c684)
- **Journalit sidebar** - Customizable sidebar making it easier to navigate throughout the plugin.
- **Search Bar** - Easily search for trades and review notes from the sidebar.

### **New Drawdown Widgets & Metrics**
![Drawdown widgets](https://github.com/user-attachments/assets/b0614747-647a-415d-ac76-1b0b0b086d0c)
- **Long and Short Drawdown Charts** - Easily compare your long and short drawdown periods.
- **More in-depth drawdpwn tooltips** - Drawdown charts now have more detailed tooltips.
- **New Drawdown Metrics** - Added Drawdown Episodes, Time in drawdown, Avg recovery time, Longest Drawdown

### **Exit Date Analytics for Swing Traders**
Choose whether dashboard and home analytics use entry date or exit date for closed trades.

### **Per-Trade Dividends**
Track multiple dividend events on a trade.

### **Percentage-Based Break-Even Thresholds**
Classify break-even using account-current-balance percentages.

### **Live Balance Calibration**
Added a separate live balance adjustment so broker-reported balance can be matched without distorting deposits or withdrawals.

### **Dashboard Percentage Metrics**
Added percentage context to dashboard Avg Win/Loss and Largest Win/Loss metrics.

### **Account Trade-Type Filter**
Added regular/backtest filtering to the Account Dashboard.

### **Contextual Onboarding Guides**
Added guided onboarding across the main Journalit views.
*Even if you are a pretty experienced user, I recommend going through these quick guides. Maybe you'll find something new.*
