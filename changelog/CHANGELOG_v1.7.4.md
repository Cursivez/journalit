# **[1.7.4] - 2026/04/30**

Privacy Mode, full French/German translations, weekly review upgrades, and a batch of dashboard/layout fixes.

### **Weekly Review Improvements**

![Key Levels](https://github.com/user-attachments/assets/32916d9a-2acf-4e68-84c8-85332c29e6d2)
More planning tools for weekly review notes.

- **Weekly Pre-Checklist** — Add a repeatable checklist to weekly reviews, with default checklist settings and per-review saved state
- **Inherited Key Levels** — Monthly levels can flow into weekly reviews, and monthly/weekly levels can flow into DRCs
- **Inline Key Level Editing** — Edit support and resistance levels directly inside review widgets
- **Default Event Notes** — Save reusable notes for events like FOMC, CPI, or NFP and auto-fill them in weekly reviews

### **Privacy Mode**

Hide sensitive trading and account values across Journalit.
_Access via Settings → General → Data Management & Privacy → Privacy Mode_
