# **[1.6.8] - 2026/03/02**
Major CSV import upgrades, new review/dashboard analytics, and reliability fixes across trade workflows.

## **Migrate your journals from Excel to Journalit!**
*Access via CSV Import → upload `.xlsx` / `.xls` files*
- **Custom Fields Mapping** - Map your imports to custom fields you created
- **Direct PnL support** — Import PnL values directly when your export provides them
- **Embedded image handling** — Review and attach images extracted from Excel exports
- **Other fees mapping** — Map/import additional fee columns during manual import

## **Scalper Demon Tracker Workflow**
Workflow support for tracking session mistakes with the new review tooling.
Use the new session mistakes widget in your DRC.

Daily workflow improvements for layout and planning.
