# **[1.7.3] - 2026/04/26**
Trade Log improvements, review template upgrades, navigation fixes, and dashboard polish.

### **Review Template Markdown Presets**
Add reusable text or questions to Markdown Zone widgets in Layout Builder.
*Access via Layout Builder → Markdown Zone*

Preset text now appears in newly created review notes, while existing answers are preserved when switching templates.

### **Trade Log Price Move Column**
See how far each trade moved from entry to exit.

### **Navigation Sidebar Recovery**
Added a dedicated command to reopen the Journalit navigation sidebar if it gets closed.
*Access via command palette → Journalit: Open navigation sidebar*

### **Navigation Behavior Fixes**
More predictable tab behavior across Journalit.
- **Sidebar setting isolation** — Sidebar tab behavior now only affects sidebar-originated navigation
- **Trade Log tab behavior** — Trades and review notes opened from Trade Log now open in new tabs as expected

### **Dashboard Metric Polish**
Dashboard percentage values now display as lightweight inline context instead of pill-style badges.

### **Home Quick Links Fix**
When all Home quick links are hidden, the empty quick-links row is now hidden in normal mode while restore guidance remains available in Customise mode.
